# Fashion Store - E-Commerce Web Application

**Role:** Backend Developer  
**Status:** Under Development (Core backend logic completed)

This repository showcases my individual backend contributions to the Fashion Store e-commerce group project. It demonstrates my ability to implement complex business logic, handle system security, and build robust features using modern Java frameworks.

## 🚀 Tech Stack
* **Core:** Java 21, Spring Boot 3.5.x
* **Database:** Microsoft SQL Server, Spring Data JPA, Hibernate
* **Security:** Spring Security, Google OAuth2
* **Payment Integration:** Momo E-Wallet API
* **Template Engine:** Thymeleaf, HTML/CSS

## 💻 My Implemented Features

### 🛒 Shopping & Checkout Processing
* **Storefront:** Designed the home page layout, built logic to load featured products, and developed the dynamic product detail page.
* **Order Placement:** Developed the Checkout page UI and implemented strict validation for checkout information.
* **Logistics:** Built the logic for users to select delivery methods during checkout.

### 🎟️ Voucher & Promotion System
* **Management:** Developed modules to manage vouchers and view voucher lists.
* **Business Logic:** Implemented the complex algorithmic logic to **Apply Voucher** securely during the checkout process based on specific conditions.

### 🔄 Customer Interaction & Admin Operations
* **Feedback System:** Built features for customers to write reviews and for administrators to manage review displays.
* **Order Lifecycle:** Implemented the logic to **Manage Return Requests**, strictly enforcing business rules (e.g., eligible order statuses).
* **System Content:** Developed a module to securely manage promotional banners on the homepage.

## 🔒 Security Note
All sensitive environment variables, including Database Credentials, Google OAuth Secrets, SMTP Passwords, and Momo API Keys, have been properly secured and excluded from this public repository adhering to security best practices.
